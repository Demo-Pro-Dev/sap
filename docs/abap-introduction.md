# What is ABAP?

ABAP (Advanced Business Application Programming) is the primary programming language used in SAP systems.

---

* https://www.youtube.com/watch?v=3-edz2POPGQ

## Key Characteristics

* Used for enterprise applications
* Strongly typed language
* Integrated with SAP database systems
* Supports object-oriented programming

---

## What to Understand

* What ABAP is used for
* Where it runs (SAP systems / cloud)
* How it interacts with databases
