# Setup Guide – Development Environment

Before the course begins, you can get a head start by preparing your development environment. This is not required but it doesn't hurt to begin familiarizing yourself with the tools you will be using. 

## Required Tools

### Eclipse IDE

https://www.eclipse.org/downloads/packages/

### ABAP Development Tools (ADT)

https://developers.sap.com/tutorials/abap-install-adt.html


## Goals

* Install Eclipse
* Install ADT plugin
* Open ABAP perspective


## Tips

* Always use the latest ADT version
* If login issues occur, try using an incognito browser session
