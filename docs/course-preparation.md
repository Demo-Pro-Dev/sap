# SAP ABAP Developer Associate Course Preparation List

In anticipation of starting the SAP ABAP Developer Associate certification training program, here are resources and references to fundamental concepts to prepare or refresh before diving into ABAP, HANA, and SAP.

> This list is intended for learners both with and without programming experience. It will be updated occasionally.

## Topics
- General Programming Fundamentals
- Object-Oriented Programming
- Database Structure & Design
- SQL Concepts + Practice
- What is ABAP?
- Object-Oriented Programming with ABAP
- About SAP
- Software Downloads / System Readiness
- SAP BTP Trial / Cloud Basics
